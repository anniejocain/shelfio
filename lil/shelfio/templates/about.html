About

Shelf.io helps you build and share shelves of books, movies, albums, and most anything on the web. Add things to Shelf.io using the Shelf.io Bookmark

Shelf.io is a project of the Harvard Library Innovation Lab.

Shelves are created using a jQuery plugin called StackView. Grab the code for StackView a GitHub. Grab the Shelf.io code at GitHub

Need more info? The next stop is to read the Common Questions page. If you still have questions, shoot the Harvard Library Innovation Lab an email.
