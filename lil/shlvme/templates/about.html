Shlv.me Button

Install the following button (drag and drop the link to your bookmarks bar).

Shlv.me

Once installed in your browser, the "Shlv.me" button lets you grab an item from Amazon, IMDB or MusicBrainz and add it to one of your shelves.

About

Shlv.me is a project of the Harvard Library Innovation Lab.

Shelves are built using a jQuery plugin called StackView. Grab the code on GitHub.
